01 — The Problem

Agents can run long. We can't measure if the long view survives.

Long-horizon failure is real and measured (METR; Illusion of Diminishing Returns, ICLR 2026) — the failures are execution, not reasoning. The long view breaks three ways, and no benchmark measures them together over a horizon:

Way 1 · Alignment

It drifts off-plan

Loses the principal's intent as the terrain shifts — OWASP ASI01 Agent Goal Hijack, the #1 agentic risk.

Way 2 · Security

The terrain hijacks it

An untrusted page, tool result, or message injects and exfiltrates — OWASP ASI02–ASI07.

Way 3 · Self-improvement

It improves the wrong way

Reward-hacking and goal drift — the Darwin Gödel Machine hacked its own reward; Anthropic shows it generalizes to sabotage.

03 — The Agent

Ranger is three models, one loop.

The Scout/Worker split is a published injection defense (Dual-LLM → CaMeL); the Dreamer adds guardrailed recursive self-improvement (Reflexion · Voyager). The novelty is the benchmark — Ranger instantiates known patterns; we measure them where no one else does.

▶ SCOUT

The long view

Maps the milestones, holds the route, re-plans when the terrain shifts, escalates the few high-risk forks.

▶ WORKER

The near terrain

Heavily sandboxed; executes steps with no authority or secrets — the only part touching untrusted data. Contains ASI02/03/05.

▶ DREAMER

Between legs

Guardrailed recursive self-improvement: purge poisoned beliefs (ASI06), self-audit vs the OWASP Top-10. Verified signal + traceable lineage so it can't reward-hack — the DGM failure.
